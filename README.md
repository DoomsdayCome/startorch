# Startorch Machine Learning Library
Startorch is a library built on top of LibTorch.

Despite being built on LibTorch, it does not aim to resemble PyTorch in any meaningful way. This project is not designed to be complete, fast, or even particularly useful. There is no goal of compatibility or feature parity, just a personal interpretation shaped by curiosity. It will most likely go nowhere, and that is entirely fine.

I like taking things apart, experimenting, and rebuilding them in my own way. Startorch exists purely as a playground to explore how AI systems work under the hood and to learn through hands on implementation. My current goal is using this self built library to solve basic machine learning problems.
